# MyScoutee ⚡

MyScoutee is an experimental Angular UI for a **group-based dating concept**, where matching is based on mutual priorities instead of endless swiping. :contentReference[oaicite:0]{index=0}

The project was largely generated by AI (~85,000 lines) and then partially refactored.
